## Features

 - [React](http://react.io)
 - [Redux](https://github.com/gaearon/redux)
 - [React Router](https://github.com/rackt/react-router)
 - [React-Hot-Loader](https://github.com/gaearon/react-hot-loader)
 - [Foundation for Apps](http://foundation.zurb.com/apps/)
